# Basic Encryption and Decryption

## Description
This project implements a basic encryption and decryption system using Caesar Cipher algorithm in Python.

## Features
- Encrypt text
- Decrypt text
- Uses shift value
- Simple Python implementation

## Technologies Used
- Python

## How to Run

```bash
python encryption.py
```

## Example

Input:
HELLO

Encrypted:
KHOOR

Decrypted:
HELLO
